Paramètres de votre RAM
Définition de la RAM allouée à votre jeu. Une quantité plus important augmentera les performances de votre jeu.
À noter qu'il faut éviter de mettre une trop grande quantité sinon cela peut provoquer des pertes de performances sur vos autres applications.

- RAM Maximale : Quantité totale de RAM allouée. Il est recommandé de ne pas dépasser 66.6% de la RAM présente sur votre PC.

- RAM Minimale : Quantité minimale de RAM allouée. Il est recommandé de ne pas mettre plus de 4 Go pour cette valeur.
Vous possédez un total de 0 Go RAM.

Paramètres de résolution
Définissez une résolution personnalisée afin d'avoir un jeu adapté à votre écran. Par défaut, le jeu démarre
en 1920 X 1080, vous pouvez modifier cette valeur pour un confort de jeu optimal.

La résolution idéale (hors plein écran) est de 1920 X 1080 (Full HD).
Une résolution trop haute (exemple: 4K) peut impacter votre carte graphique !
Par défaut
Paramètres de téléchargement
Détermine le nombre de fichiers pouvant être téléchargés simultanément lors de la mise à jour ou du téléchargement des ressources.

Veuillez noter que la modification de ce paramètre peut affecter les performances de téléchargement.
Il est recommandé de ne pas dépasser les limites recommandées par défaut,
sauf si vous rencontrez des problèmes spécifiques de connectivité.
